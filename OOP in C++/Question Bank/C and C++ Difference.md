# ➕ Additional 10 Comparison Points Between C and C++

## 11️⃣ File Handling
| Feature | C | C++ |
|--------|---|------|
| File Handling | `fopen`, `fread`, `fwrite` | File streams (`ifstream`, `ofstream`) |
| Ease of Use | More manual | More user-friendly |

---

## 12️⃣ Standard Template Library (STL)
| Feature | C | C++ |
|--------|---|------|
| STL | ❌ Not available | ✔ Available (vector, list, map, algorithms) |
| Level of abstraction | Low | High |

---

## 13️⃣ Function Arguments
| Feature | C | C++ |
|--------|---|------|
| Default Arguments | ❌ Not supported | ✔ Supported |
| Pass-by-reference | ❌ Not directly | ✔ Using `&` |

---

## 14️⃣ Security (Type Safety)
| Feature | C | C++ |
|--------|---|------|
| Type Checking | Less strict | More strict |
| Strong Type Casting | ❌ No | ✔ Yes (`static_cast`, `dynamic_cast`) |

---

## 15️⃣ Constructors & Destructors
| Feature | C | C++ |
|--------|---|------|
| Constructor | ❌ Not available | ✔ Yes |
| Destructor | ❌ Not available | ✔ Yes |

---

## 16️⃣ Inline Expansion
| Feature | C | C++ |
|--------|---|------|
| Inline Functions | ❌ Not supported | ✔ Supported (`inline` keyword) |
| Macro Replacement | ✔ `#define` | ✔ But inline preferred |

---

## 17️⃣ Encapsulation Level
| Feature | C | C++ |
|--------|---|------|
| Access Specifiers | ❌ Not available | ✔ `private`, `public`, `protected` |
| Data Hiding | Very limited | Fully supported |

---

## 18️⃣ polymorphism
| Feature | C | C++ |
|--------|---|------|
| Compile-time | ❌ No | ✔ Supported (Function overloading) |
| Runtime | ❌ No | ✔ Supported (Virtual functions) |

---

## 19️⃣ Templates / Generic Programming
| Feature | C | C++ |
|--------|---|------|
| Templates | ❌ Not available | ✔ Supported (`template<class T>`) |
| Generic Code | Hard | Easy & powerful |

---

## 20️⃣ Object Lifetime & Memory Model
| Feature | C | C++ |
|--------|---|------|
| Object Lifetime | No objects | Constructor/destructor manage lifetime |
| Memory Safety | Manual | More automatic (RAII) |

---

# ⭐ Combined Summary (20 Points Now!)
C is a procedural, low-level, fast language used mainly for system programming.  
C++ extends C with object-oriented features, abstraction, templates, STL, and better memory management — making it ideal for complex applications and software engineering.
